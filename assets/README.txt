Drop your app icon and splash image here:
  icon.png        (1024x1024)
  splash.png      (1284x2778 recommended)
  adaptive-icon.png (Android, 1024x1024)

Then reference them in app.json under expo.icon, expo.splash.image,
and expo.android.adaptiveIcon.foregroundImage.

The project runs without these — Expo uses defaults — so you can start
immediately and add branded assets later.
